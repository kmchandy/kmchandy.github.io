Example: Flight Formation

This module describes the proof of a simple concurrent system in which a collection of agents in space, move towards a formation, even though each agent can only communicate with its neighbors.

Problem Description

The problem deals with agents moving in 3-D space; however, since the essential ideas of the solution are clear for the case of 2-D we restrict consideration to 2-D space.

Agents \(0\) and \(N\) are stationary. The action of agent \(j\) for \(0 < j < N\) is as follows. If the distance from the agent's location to the midpoint of its neighbors is more than or equal to \(\epsilon\) then the agent moves to the midpoint; otherwise, the agent does nothing.

The action is specified by the following program segment. Let \(l\) be the vector of agents' locations where \(l[j]\) is the location of the \(j\)-th agent. 

    mid = (l[j-1] + l[j+1])/2.0
    if |l[j] - mid| >= epsilon:
         l[j] = mid

The program segment is equivalent to the guarded command:

\( |l[j] - mid| \geq \epsilon \quad \rightarrow \quad l[j] = mid \)

Specification

The desired terminal or goal state is specified by a predicate \(goal\) which specifies that each agent's location should be \(\delta\)-close to the midpoint of the locations of its neighbors. \(goal\) is defined as:

\( \forall 0 < j < N : \) \( \quad |l[j] - mid | < \epsilon \)

where \(mid = (l[j-1]+ l[j+1])/2 \)

The figure below shows the desired goal state.

The specification is that the program must terminate, and must terminate at a goal state.

Exercise: Prove Correctness

Answer

Next, we give the structure of a proof. The challenge is to find a variant function. After you find a variant function the proof follows from elementary geometry.

Variant Function

Let \(f(l)\) be a variant function defined as the sum of the square of the lengths between neighboring agents.

\( f(l) = \sum_{j=1}^{N} ((x_{j} - x_{j-1})^{2} + (y_{j} - y_{j-1})^{2}) \)

Proof Obligation

We need to show that the execution of any guarded command with a True guard reduces the value of the variant function by at least a positive constant \(\delta\). This variant function is bounded below by \(0\). So, if the value of the variant function in a state is \(K\) then the program terminates in at most \(K/\delta\) steps from this state.

A little bit of high-school geometry on triangles shows that execution of any guarded command with a True guard changes \(f(l)\) by at least a positive constant \(\delta\). We leave it to you to determine \(\delta\). The largest value of \(\delta\) is immaterial; all you need to do is show that there exists a positive value. (Hint: \(\delta\) depends on \(\epsilon\) and the initial configuration.)

Termination Condition

Our proof obligation is to show that if all guards evaluate to False then\(goal\) holds. This is trivial.
