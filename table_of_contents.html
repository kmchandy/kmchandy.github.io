Section: Graphs

This section describes concepts in terms of labeled directed graphs. This section has three main modules with supporting modules containing self tests, exercises and explorations: These modules are safety, progress, and properties of temporal operators. In addition, the section has two exercise modules which you can view as exercises that you can skim through. Live discussions on this section spend little time on exercise module.

Safety

This module introduces all the concepts that we use in this course to prove that systems are safe. These concepts are described in terms of directed graphs.

Progress

This module introduces all the concepts that we use in this course to prove that systems progress towards their goals. These concepts are described in terms of labeled graphs.

Properties of Temporal Operators

This module defines key properties of the relation, leads-to (or \(\leadsto\)). We use these properties to prove progress of every system that we study in this course. These properties are defined in terms of paths and sets of vertices in labeled graphs.

Exercise Module: Properties of Relations

This module is an exercise module. It gives properties of relations that are similar to properties of \(\subseteq\). You can treat this module as an exercise in which you find properties of relations that we use in this course.

Exercise Module: Composition of Operators

This module is an exercise module. It gives examples of composition of functions and relations that are defined in the earlier modules. You can view this module as an exercise in which you identify the properties of different compositions.
