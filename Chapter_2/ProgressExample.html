Examples of Fair Paths

  1. \( 3 \stackrel{a}{\rightarrow} 4 \stackrel{b}{\rightarrow} 3 \stackrel{a}{\rightarrow} 4 \stackrel{b}{\rightarrow} 3 \stackrel{a}{\rightarrow} 4 \stackrel{b}{\rightarrow} 3 \ldots \)
    This path cycles between vertices \(3\) and \(4\) traversing edges labeled \(a\) and \(b\) for ever.
  2. \( 8 \stackrel{a}{\rightarrow} 9 \stackrel{a}{\rightarrow} 8 \stackrel{a}{\rightarrow} 9 \stackrel{c}{\rightarrow} 1 \stackrel{a}{\rightarrow} 1 \stackrel{b}{\rightarrow} 1 \stackrel{a}{\rightarrow} 1 \stackrel{b}{\rightarrow} 1 \ldots \)
    This path reaches vertex \(1\) and loops around it traversing edges labeled \(a\) and \(b\) for ever.

Examples of paths that are not fair

  1. \( 8 \stackrel{b}{\rightarrow} 8 \stackrel{a}{\rightarrow} 9 \stackrel{a}{\rightarrow} 8 \stackrel{a}{\rightarrow} 9 \stackrel{a}{\rightarrow} 8 \stackrel{a}{\rightarrow} 9 \stackrel{a}{\rightarrow} 8 \ldots \)
    This path traverses an edge labeled \(b\) once and thereafter traverses only edges labeled \(a\). This path is not fair because it only traverses infinitely-often label \(b\) only a finite number of times.
  2. \( 6 \stackrel{a}{\rightarrow} 5 \stackrel{b}{\rightarrow} 3 \stackrel{b}{\rightarrow} 0 \stackrel{b}{\rightarrow} 5 \stackrel{b}{\rightarrow} 3 \stackrel{b}{\rightarrow} 0 \stackrel{b}{\rightarrow} 5 \stackrel{b}{\rightarrow} 3 \stackrel{b}{\rightarrow} 0 \ldots \)
    This path traverses an edge labeled \(a\) once and thereafter traverses only edges labeled \(b\). This path is not fair because it only traverses infinitely-often label \(a\) only a finite number of times.

An example of \(P \leadsto Q\)

is: \( \{8, 9\} \leadsto \{1, 2\} \)
A fair path that enters any vertex in \(\{8, 9\}\) cannot remain in \(\{8, 9\}\) for ever because the path will traverse infinitely-often label \(b\) which takes the path out of \(\{8, 9\}\). Transitions out of \(\{8, 9\}\) are to \(\{1, 2\}\).
  1. \( \{1, 2\} \leadsto \{1, 2\} \)
    There is a path, with no hops, from every vertex to itself. So \(v\) is in \(E(\{v\})\) for all vertices \(v\).
  2. \( \{8, 9\} \leadsto \{1, 2\} \)
    A fair path that enters any vertex in \(\{8, 9\}\) cannot remain in \(\{8, 9\}\) for ever because the path will traverse infinitely-often label \(b\) which takes the path out of \(\{8, 9\}\). Transitions out of \(\{8, 9\}\) are to \(\{1, 2\}\).
  3. \( \{8\} \leadsto \{1, 2\} \) and\( \{9\} \leadsto \{1, 2\} \)
    Follows from the discussion above.
  4. \( \{10\} \leadsto \{9\} \)
    A fair path that enters \(\{10\}\) cannot remain in \(\{10\}\) for ever because the path will traverse infinitely-often label \(a\) which takes the path out of \(\{10\}\). Transitions out of \(\{10\}\) are to \(\{9\}\).
  5. \( \{10\} \leadsto \{1, 2\} \)
    A fair path that enters \(\{10\}\) later enters \(\{8, 9\}\), and a path that enters \(\{8, 9\}\) later enters \(\{1, 2\}\).
  6. \( \{1, 2, 8, 9, 10\} \leadsto \{1, 2\} \)
    Follows from the previous results.

An examples of \(\neg (P \leadsto Q)\)

is: \( \neg (\{7\} \leadsto \{1, 2\}) \)
There is a fair path from \(7\) throug \(6\) that never visits \(\{1, 2\}\).
  1. \( \neg (\{7\} \leadsto \{1, 2\}) \)
    There is a fair path from \(7\) throug \(6\) that never visits \(\{1, 2\}\).
  2. \( \neg (\{0, 3, 4, 5, 6\} \leadsto \{1, 2\}) \)
    There exist fair paths from these vertices that never visit \(\{1, 2\}\).

An example of Eventually: E(P)

is: \( E({1, 2}) = \{1, 2, 8, 9, 10\} \) because all paths that start at vertices in \{1, 2, 8, 9, 10\} and that traverse labels \(a\) and \(b\) infinitely often, visit vertices \(1\) and \(2\).
  1. \( E({1, 2}) = \{1, 2, 8, 9, 10\} \)
    From the earlier discussion, \( \{1, 2, 8, 9, 10\} \leadsto \{1, 2\} \) and \( \neg (\{0, 3, 4, 5, 6, 7\} \leadsto \{1, 2\}) \)
  2. \( E({0}) = \{0\} \)
    Every path from \(0\) visits \(0\), by definition; so, all fair paths from \(0\) visit \(0\). So, \(0\) is in \(E({0})\). One can construct fair paths from all the vertices other than \(0\) that never visit \(0\).
