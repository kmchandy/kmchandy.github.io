Timelines

A timeline diagram depicts the evolution of a single trajectory over time. In a timeline diagram, the horizontal (i.e. x-) axis represents time. Each agent has its own timeline which shows the evolution of states of that agent. An agent's timeline is represented by a horizontal line; the state of an agent \(B\) at time t is represented by the point, at time \(t\), on \(B\)'s line. A state transition of an agent is called an event at that agent.

Example

In the figure the blue horizontal lines are the timelines for agents \(A\) and \(B\). The blue circles numbered 1, 3, 5 on the timeline for agent \(A\) represent events at agent \(A\); likewise, blue circles numbered 2, 4 on the timeline for agent \(B\) represent \(B\)'s events. An event can be an internal state transition, which is one without receiving messages. An example of an internal event is event 1 on the timeline for agent \(A\). Events 2, 3, 4, 5 are receive-message events.

Message Timelines

A message sent by an agent \(A\) at time \(t\) and received by agent \(B\) at time \(t'\) is represented by a straight line starting at point \(t\) on \(A\)'s timeline and ending at point \(t'\) on \(B\)'s timeline; this line is the timeline for the message. In the diagram, agent \(A\) sends messages \(m_{0}\) and \(m_{1}\), while agent \(B\) sends messages \(r_{0}\) and \(r_{1}\).

Channel States in Timeline Diagrams

The state of a channel \(c\) at time \(T\) is the sequence (in time order) of messages that are sent along \(c\) before \(T\) and that are received after \(T\).

Example

The next diagram shows how the global state of the system --- the states of all agents and channels --- at time \(T\) are represented in the timeline diagram. The diagram shows timelines for agents \(A, B, D\). Agent \(A\) sends messages \(m_{0}\) along a channel \(c'\) and \(m_{1}\) along a different channel \(c\) at event 1. When agent \(D\) receives \(m_{0}\) at event 2 it sends message \(r_{0}\) to \(A\). When agent \(A\) receives \(r_{0}\) at event 3 it sends message \(m_{2}\) to \(B\) along channel \(c\).

In the diagram, the state of \(c\) is the sequence of \(c\)-message lines that are cut by the vertical line at \(T\); these are the message lines for \(m_{1}\) and \(m_{2}\) in that order. The state of an agent \(A\) at \(T\) is, likewise, the state on \(A\)'s timeline at the point where the timeline is cut by the vertical line at \(T\). The global state of the system at T is obtained from the points at which agent timelines and message lines are cut by the vertical line at \(T\).
