Review of Basics
Nondeterministic Iteration: Self Test

Question 1

A do-od loop L satisfies the Hoare triple:
{P} L {Q},
for given predicates P, Q. The loop consists of 2 guarded commands: 

do
   g0  ->  c0
   g1  ->  c1
od
A while loop L' is as follows: 
while (g0 OR g1):
   if g0 then c0
   if g1 then c1
Is the following true or false? {P} L' {Q}

Answer to Question 1

True

Every execution of the deterministic loop L', starting in a state in which P holds, is also an execution of the nondeterministic loop L, starting in a state in which P holds, where the guarded commands of L are executed repeatedly in order: (1) execute the first guarded command and then (2) execute the second guarded command.

So, if every execution of L ends in a state in which Q holds then every execution of L' also ends in a state in which Q holds.

Question 2

A do-od loop L with the following two guarded commands satisfies the Hoare triple {P} L {Q}. 
do
   g0  -> c0
   g1  -> c1
od
The do-od loop L' is as follows: 
do
   g0 -> c0
   g1 -> c1
   (g0 AND g1)  -> c0
   (g1 AND g0)  -> c1
od

Is the following statement true or false? L' satisfies the same Hoare triples as L, i.e {P} L' {Q} holds.

Answer to Question 2

True

Any path through the state-transition diagram from any state in L' is also a path from the same state in L. Why?

Let the guarded commands in loop L' be indexed 0, 1, 2, 3, in order. Given an execution path for L' we will create the same path for L. Order the execution of guarded commands in L as follows.

  1. guarded command g0 -> c0
  2. guarded command g1 -> c1
  3. if g0 AND g1 hold then execute guarded command g0 -> c0, otherwise skip
  4. if g1 AND g0 hold then execute guarded command g1 -> c1, otherwise skip
