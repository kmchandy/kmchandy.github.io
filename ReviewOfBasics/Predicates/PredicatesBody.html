Predicates on States

This module is a review of predicates on states of programs.

The state of a program is given by the values of its variables. We will use set operations, such as union and intersection, on sets of states. For this course, a predicate is a function from states of a program to the Boolean constants \(\{True, False\}\). For example, for a program with variables x, y, the predicate x > y is true in the state (x, y) = (1, 0) and is false in the state (x, y) = (1, 1).

Set of elements associated with a predicate

Associated with a predicate \(P\) is a set of all the states of a program for which \(P\) holds; this set is called the extension of predicate \(P\) . Disjunction (or), conjunction (and) and negation (not) of predicates correspond to the union intersection, and complement, respectively, of the corresponding sets of states. The extensions of the predicates True and False are the universal and empty sets, respectively.

For convenience, we use predicates more often than sets of states.
