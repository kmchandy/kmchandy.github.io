Termination Detection

A distributed computation has terminated when all agents are idle and all channels are empty. A termination detection algorithm is executed by the operating system to determine whether a client computation has terminated. Detection algorithms are based on global snapshots.

Problem Definition

A computation terminates in states in which all channels are empty and all agents are waiting to receive messages. A channel \(C\) is empty exactly when \(\; C_{s} = C_{r}\), where \(C_{s}\) and \(C_{r}\) are the numbers of messages sent and received (respectively) on \(C\). So:

A computation is in a terminated state exactly when:
(1) all agents are waiting to receive messages and
(2) For all channels \(C\): \(\; C_{s} = C_{r}\).

How Should You Approach the Problem?

Use the General Method for Detecting Stable Properties

The OS monitors a system by taking repeated global snapshots. If a global snapshot is a state in which the computation has terminated then the computation has indeed terminated. And if a computation has terminated then repeated snapshots ensure that a snapshot is taken after computation terminates.

An Alternate Method

The algorithm uses an OS agent called an observer. Let \(C_{s}\) and \(C_{r}\) be the numbers of messages sent and received (respectively) on channel \(C\). Each time an agent completes executing a receive it sends a message to an observer. This message contains \(C_{s}\) for each of that agent's outgoing channels \(C\), and \(C_{r}\) for each of the agent's incoming channels \(C\).

The agent keeps only the latest message that it receives from each agent. Let \(C_{s}^{*}, C_{r}^{*}\) be the last value of \(C_{s}, C_{r}\) (respectively) that the observer has received. Initially \(C_{s}^{*} = 0, C_{r}^{*} = -1\) for all channels \(C\) so as to ensure \(C_{s}^{*} \neq C_{r}^{*}\).

The observer detects computation has terminated if: for all channels \(C\): \(C_{s}^{*} = C_{r}^{*}\).

Proof of Correctness

Define the local snapshot of an agent as the point at which it sends \(C_{s} = C_{s}^{*}\) for each of its output channels \(s\), and \(C_{r} = C_{r}^{*}\) for each of its input channels \(r\).

The local snapshots correspond to a global snapshot exactly when all messages received in the past are sent in the past which is the case when \(C_{s}^{*} = C_{r}^{*}\).
