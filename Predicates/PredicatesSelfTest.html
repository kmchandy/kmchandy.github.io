Predicates: Self Tests

This exercise deals with predicates on students at a university. Let \(CS, Sp, IQ\) be predicates that hold for students in CS, who play a sport, and who are smart, respectively. Draw Venn diagrams that illustrate the following predicates, where the sets in the diagrams are extensions of the predicates.
  1. \(CS \wedge IQ\)
  2. \(CS \vee IQ\)
  3. \((\neg CS) \wedge (\neg IQ)\)
  4. Note square brackets.

    \([IQ \equiv CS]\)

  5. No square brackets.

    \(IQ \equiv CS\)

  6. Note the square brackets:

    \([CS \Rightarrow IQ]\)

  7. No square brackets; so, this is a predicate, not a Boolean.

    \(CS \Rightarrow IQ\)

  8. The following formula is a Boolean value because both sides of the implications are Booleans. Is the formula True or False?

    \([IQ \equiv (CS \wedge Mu)] \Rightarrow [IQ \Rightarrow CS] \)

Predicates: Self Tests Answers

  1. \(CS \wedge IQ\)

    Intersection of the extensions of CS and IQ.

  2. \(CS \vee IQ\)

    3. Union of the extensions of CS and IQ.

  3. \((\neg CS) \wedge (\neg IQ)\)

    4. The extension of \(\neg CS\) is the universal set excluding the extension of CS. Similarly for \((\neg IQ)\).

  4. Note square brackets.

    \([IQ \equiv CS]\)

    Because of the square brackets, this term is a Boolean and not a predicate. It is either the Boolean True which means it holds for this universal set or it is the Boolean False. It is not a function from an element to the Booleans.

    This Boolean is True exactly when the extensions of \(IQ\) and \(CS\) are identical. So, this Boolean is True for our set of students exactly when a student is smart if and only if the student is in CS.

    Question:
  1. No square brackets.

    \(IQ \equiv CS\)

Answer

This is a predicate. It is a function from elements of the set to the Booleans.

Consider a student Maya who is smart and in CS. This predicate, applied to Maya, returns the Boolean "Maya is smart" = "Maya is in CS", which is True.

Consider another student, Stupid, who is not smart and is not in CS. This predicate applied to Stupid is the Boolean True because "Stupid is smart" = "Stupid is in CS", since both sides of the equality are False.

Look at Einstein who is smart but is not in CS. The predicate applied to Einstein is False.

The extension of \(IQ \equiv CS\) is the union of (1) the intersection of the extensions of \(IQ\) and \(CS\), and (2) the intersection of the extensions of \(\neg IQ\) and \(\neg CS\).

    Question:
  1. Note the square brackets:

    \([CS \Rightarrow IQ]\)

  2. No square brackets; so, this is a predicate, not a Boolean.

    \(CS \Rightarrow IQ\)

Answer

The Venn diagram on the left shows \([CS \Rightarrow IQ]\). This is a Boolean: It is either True for the universal set in question or it is False. It is not a function from elements of the set to the Booleans.

It is True for the universal set when the extension of \(CS\) is a subset of the extension of \(IQ\). It says that every student in CS is smart.

The Venn diagram on the right shows \(CS \Rightarrow IQ\) without the square brackets. This is a function from elements of the universal set to the Booleans.

Apply this function to Stupid who is not in CS and not smart. The antecedent "is in \(CS\)" is False, and so it doesn't matter what the consequence is: This function applied to Stupid is True.

Applying this function to student X who is in CS and is not smart returns False.

The extension of this predicate \(CS \Rightarrow IQ\) is the union of (1) of (NOT CS) and (2) IQ.

Question 8

Yes, the formula is True because:

\( [CS \wedge Mu \quad \Rightarrow \quad CS] \)
