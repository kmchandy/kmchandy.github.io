Review of Predicates

This module is a review of predicates. Skim through this module to refresh your memory or skip it.

A predicate on a set \(V\) of elements is a function from subsets of \(V\) to the Boolean constants \(\{True, False\}\).

\(V\) is called the universal set.

Let \(V\) be a set of the following three students:

  1. name: Maya, age: 19, major: CS
  2. name: Liu, age: 20, major: Physics
  3. name: Eve, age: 21, major: CS

\(age > 19\) is a predicate because it is a function from elements of the set to \(\{True, False\}\). The result of applying this function to (Maya, 19, CS) is False, while applying this function to (Liu, 20, Physics) returns True.

Extension of a Predicate

A predicate \(R\) defines a set \(\{v | R(v) \}\) called its extension. An extension of a predicate is the set of elements for which the predicate holds. In the example, the extension of the predicate \(age > 19\) is the set \(\{ (Liu, 20, Physics), (Eve, 21, CS) \}\).

Predicates can be composed using Boolean operators. Set union corresponds to predicate disjunction (or), and set intersection to predicate conjunction (and). For a predicate \(X\), let \(e(X)\) denote its extension. Here are examples of extensions.

Examples

\( e((age > 19) \wedge (major = CS)) \; = \; e(age > 19) \cap e(major = CS) \)

So, \( e((age > 19) \wedge (major = CS)) \; = \; \{(Eve, 21, CS)\} \)

\( e((age > 20) \vee (major = physics)) = \{(Liu, 20, Physics), (Eve, 21, CS)\} \)

\( e(\neg (major = physics)) = \{(Maya, 19, CS), (Eve, 21, CS)\} \)

True, can be viewed as a predicate whose extension is the entire set \(V\) over which predicates are defined. Similarly, False can be viewed as a predicate whose extension is the empty set.

Implication

\(P \Rightarrow Q\) is a predicate which is equivalent to \(\neg P \vee Q\).

Example

\((major = CS) \; \Rightarrow \; (age < 20) \) is a predicate which when applied to Maya is True; when applied to Liu is also True because the antecedent \(major = CS\) doesn't hold for Liu; and when applied to Eve is False because the antecedent is True whereas the consequent, \(age < 20\), is False.

Formula as a Predicate or a Boolean

We may want to use a formula, \(P\), as a predicate - i.e., a function - or as a boolean. We use Edsger W. Dijkstra's notation of square brackets, \([P]\), to refer to the boolean: \(P\) is universally True. \(P\) is universally True means that the the extension of \(P\) is the universal set, \(V\).

Example

\([(major = physics) \vee (age = 19) \vee (age = 21)] \)

In our example this predicate holds for Maya, Liu, and Eve - it is universally True. Without the square brackets,

\((major = physics) \vee (age = 19) \vee (age = 21) \)

is a predicate that we can apply to each element --- Maya, Liu, Eve. Applying it to each element returns True.

With the square brackets we get a Boolean which is not a function from elements to the Booleans. It is a property of the universal set.

Equivalence

\(P \equiv Q\) is a predicate which holds exactly when both \(P\) and \(Q\) are True, or both are False.

\( (P \equiv Q) \quad \equiv \quad (P \wedge Q) \vee (\neg P \wedge \neg Q) \)

We almost always use equivalence in formula that are universally True, denoted by \([P \equiv Q]\). This means that \(P\) and \(Q\) are identical.

Example

Let \(odd\) be a function on integers that returns True exactly when the integer is an odd number.

\( odd(age) \; \equiv \; (major = CS) \)

is a predicate which when applied to Maya and Eve return True because both sides of the equivalence are True; and when applied to Liu returns True because both sides of the equivalence are False. Since the extension of this predicate is the universal set, we write:

\([odd(age) \; \equiv \; (major = CS)]\)

Predicates on States

In this course we will be using predicates on states. Proofs using predicates can be converted to proofs using sets of states by treating each predicate as its extension.
