Distributed Algorithms

About the Website

This website contains descriptions of distributed algorithms. The descriptions include videos, self tests, and examples. Discrete-state and continuous-state, continuous-time algorithms are included. The description of each algorithm is self contained. Algorithms, programs and simulations will be added. A subset of the material was used in a course for third-year undergraduates at Caltech.
