Distributed Algorithms Contents Index
This website shows that algorithms used in distributed systems are based on a few simple ideas.

Distributed systems consists of agents that communicate with each other by sending and receiving messages. Diverse applications are based on message-passing agents. Cryptocurrencies are based on agents that both compete and collaborate by sending and receiving messages. The power grid is a distributed system with information from power generators, energy markets, and substations. Agents in multiplayer games communicate by sending messages.

The website has implementations of algorithms in Python and other languages. This website has been used in a course taken by undergraduates in their first or second year. Elementary graph theory is the only prerequisite.
