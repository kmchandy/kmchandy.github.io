Distributed Algorithms Contents Index
This website shows that algorithms used in distributed systems are based on a small number of simple concepts.

The website contains descriptions of a collection of algorithms that do not have to be read in order. Each algorithm description is concise but has enough detail to allow readers to implement programs and construct formal proofs.

The material in this website has been used in an introductory undergraduate course. Elementary graph theory is the only prerequisite.

K. Mani Chandy, Emeritus Simon Ramo Professor, California Institute of Technology
